# 示例照片素材目录

这里放"TA 可以是任何样子"那一屏拍立得卡片的示例图。

## 期望文件名（在 `PossibilitiesSection.tsx` 里引用）

| 文件名             | 用途             | 建议尺寸           |
| ------------------ | ---------------- | ------------------ |
| `selfie.jpg`       | 自拍 / 个人照    | 正方形 ~ 400×400+  |
| `family.jpg`       | 家人 / 朋友合照  | 正方形 ~ 400×400+  |
| `pet.jpg`          | 宠物（猫/狗等）  | 正方形 ~ 400×400+  |
| `character.jpg`    | 二次元 / 角色    | 正方形 ~ 400×400+  |
| `photo.jpg`        | 任意照片         | 正方形 ~ 400×400+  |

## 几条建议

1. **正方形优先**：拍立得卡片的图片区是 1:1，非正方形会被 `object-fit: cover` 裁剪
2. **尺寸不用太大**：最终展示宽度只有 ~120px，400×400 已经足够清晰，文件别太大
3. **`.jpg` 或 `.png` 都行**：jpg 文件更小，更适合照片
4. **没图也没关系**：`onError` 会自动降级到 emoji 占位（已写好兜底）

## 想换文件名？

直接改 `src/components/PossibilitiesSection.tsx` 顶部的 `EXAMPLES` 数组即可。
